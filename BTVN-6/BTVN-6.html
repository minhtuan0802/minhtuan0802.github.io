Nền tảng lập trình

Java Springboot Fullstack

Lập trình Fullstack từ Frontend tới Backend với Java, khóa học duy nhất bao gồm cả kiến thức về cấu trúc dữ liệu và giải thuật

4.5 tháng - 36 buổi

Web Frontend + ReactJS

Tập trung vào lập trình Web Frontend, bao gồm cả thiết kế UI/UX và các framework được sử dụng để xây dựng giao diện hot nhất hiện nay như ReactJS

4.5 tháng - 36 buổi

Mobile React Native

React Native tạo ra bởi Facebook, được sử dụng để lập trình ứng dụng cho cả 2 nền tảng di động phổ biến nhất hiện nay là iOS và Android

5 tháng - 60 buổi

DevOps

DevOps là một văn hóa làm việc kết hợp giữa kỹ sư phát triển phần mềm (dev) với bộ phận operator (kỹ sư hệ thống, nhân viên bảo mật, kỹ sư mạng, kỹ sư hạ tầng, …) nhằm mục đích rút ngắn vòng đời phát triển sản phẩm

5 tháng - 42 buổi

Software Testing

Kiểm thử là một quy trình quan trọng trong lĩnh vực phát triển phần mềm, giúp đảm bảo chất lượng trước khi khách hàng tiếp nhận sản phẩm. Nhu cầu nhân lực trong lĩnh vực này rất lớn, và mức lương cũng cao tương đương với lập trình viên.
